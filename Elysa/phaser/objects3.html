Objects: Fixing A Roller Coaster

Even though the Big Dipper 2 is safe, it's not as popular a ride as you want to build. You decide to draw up plans for a new roller coaster that you want to be safe and popular. However, the requirements for a safe roller coaster have changed, and you also don't know what makes a ride popular. You have to design a roller coaster that (1) is safe, and (2) has a happiness rating of 95 or more.

A roller coaster must have the following properties:
  - rideType
  - name
  - minRiderHeightInInches
  - numLoops
  - color
  - material (either "wood" or "steel")

Each property value can either satisfy or not satisfy the new safety requirements, and different values can increase or decrease the happiness rating. Declare an object variable rollerCoaster with all the necessary properties, and find a combination of values that make it safe and fun.

Hint

As a reminder, you want to declare rollerCoaster as an object. You can construct an object with the syntax:
    var object = {
        property1: value1,
        property2: value2
    };

Success!

You designed a roller coaster that is safe and popular, and you've learned all about Javascript objects!
