Objects: Fixing A Roller Coaster

The sign didn't help. You heard that visitors are worried about the safety of all the park's rides.

You have been told that a roller coaster is safe if:
  - its minimum rider height is at least 52 inches
  - it has 3 or fewer loops
  - its speed is 100 mph or less if it is wooden, or 125 mph or less if it is steel

Right now, the Big Dipper is not safe. Create a new variable, bigDipper2, which has the same properties as bigDipper but satisfies the safety requirements. Run your code when you're ready for the safety test.

Hint

Try creating the bigDipper2 object below the object bigDipper. The construction should look almost the same, you just have to make sure bigDipper2 satisfies the safety requirements.

Success!

Nice work creating a safe roller coaster. bigDipper and bigDipper2 are both objects, which you construct with the syntax:
    var object = {
        property1: value1,
        property2: value2
    };
