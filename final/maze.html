Control Flow: Crazy Maze

Welcome to the Crazy Maze.js!

But for some reason, it looks more like a big field than a crazy maze. All the walls are gone...
But with your knowledge of Javascript and for loops and Math.random() and conditionals, you can make your own maze!

You have two different types of walls at your disposable, vert and horiz (vertical and horizontal). Look at the code to the right to see how to make a code. Then uncomment the lines and press Run Code to see the wall appear.

Be creative in your maze building, and challenge your friend to complete it! To restart it, refresh the page, and click "Need a hint?" if you get stuck!

Hint

Now that you know about for loops, you can make multiple walls at once.

for (var i = 0; i < 10; i++) {
// insert code here
}

Now that you know about Math.random(), you can randomly put multiple walls around your maze. And since you know about if statements, you can use randomness to decide what type of walls you can make.

for (var i = 0; i < 10; i++) {
if (Math.random() > .5) {
// insert code here
}
else {
// insert code here
}
}
